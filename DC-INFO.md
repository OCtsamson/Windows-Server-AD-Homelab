# What is a Domain Controller and Why Should I Promote My Server?

## Domain Controllers manage authentication, users, devices, policies, and security for a Windows Domain

### Group Policy Management 
- Allows setting enforcement across all computers in the domain 
  - Password requirements
  - Software Deployment 
  - Printer and Drive Mapping
  - Windows Update Policies

### Device Management 
- Computers that join the domain can be managed centrally
  - Admins can reset passwords
  - Disable accounts
  - Add/remove devices
  - Control permissions

### Shared Resources and Permissions 
- Permissions can be assigned to groups instead of individual users
  - File servers
  - Shared folders
  - Printers
